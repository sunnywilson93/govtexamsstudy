### 1. Percentage

**5 Core Concepts**
1. Percent means "per hundred" — any percentage is a fraction with denominator 100; converting fluently between %, fraction, and decimal is the foundation
2. Percentage change formula: `Change% = (Change / Original) × 100` — always divide by the original (base), not the new value
3. Successive percentage changes: applying two changes `a%` and `b%` yields net `= a + b + (ab/100)%` — the cross-product term `ab/100` is the key insight
4. Percentage point vs. percentage: if something goes from 20% to 30%, the increase is 10 percentage points but 50% in relative terms — distinguish carefully in DI questions
5. Base equivalence: 20% of X = X% of 20 — this symmetry helps when the other value is easier to compute

**5 Tricks**
1. **Formula Shortcut — Successive change:** Net% = a + b + ab/100 (skip step-by-step, plug directly)
2. **Formula Shortcut — Reverse percentage:** If a value after P% increase is N, original = N × 100/(100+P); after P% decrease = N × 100/(100−P)
3. **Mental Math — Finding 1%:** Divide by 100 (move decimal two places left); then multiply to find any %; e.g. 7% of 340 = 3.40 × 7 = 23.8
4. **Mental Math — Finding 15%:** = 10% + 5% = 10% + half of 10%; faster than direct computation
5. **Mental Math — 12.5% = 1/8:** Memorize common fraction equivalents: 12.5%=1/8, 16.67%=1/6, 33.33%=1/3, 66.67%=2/3, 37.5%=3/8

**5 Problem Types**
1. Find X% of a number (direct application)
2. Find the original number when a percentage of it is given
3. Population / salary successive growth over multiple years
4. Price increase/decrease — find new price or original price
5. Two successive percentage changes — find net percentage change

### 2. Profit, Loss & Discount

**5 Core Concepts**
1. Base is always Cost Price (CP): `Profit% = (Profit / CP) × 100`; `Loss% = (Loss / CP) × 100` — SP and discount are never the base
2. Selling price formulas: `SP = CP × (100 + P%) / 100` for profit; `SP = CP × (100 − L%) / 100` for loss
3. Marked Price (MP) and Discount: `SP = MP × (100 − Discount%) / 100`; discount is always on MP, not CP
4. Dishonest dealer trick: using faulty weights to gain extra profit; if dealer uses `x g` weight instead of 1000 g, profit% = `(1000 − x) / x × 100`
5. Profit% when cost includes overhead: total CP = purchase price + overhead expenses; compute profit% on this total CP

**5 Tricks**
1. **Formula Shortcut — CP from SP and P%:** `CP = SP × 100 / (100 + P%)`; memorize this, don't re-derive
2. **Formula Shortcut — Two successive discounts:** Net discount = a + b − ab/100 (same as successive % formula)
3. **Formula Shortcut — Equal items at different prices:** If n items bought at same total as n items sold, P% = difference ratio × 100
4. **Mental Math — Profit/Loss in 1 step:** If SP = 1.25 × CP → 25% profit; if SP = 0.80 × CP → 20% loss; read the multiplier directly
5. **Formula Shortcut — Faulty weight gain:** Profit% = (True weight − False weight) / False weight × 100

**5 Problem Types**
1. Find CP/SP given the other and P% or L%
2. Find overall P% or L% when articles bought at different rates and sold at one rate
3. Marked Price, discount percentage, and final SP problems
4. Two successive discounts vs. single equivalent discount
5. Dishonest dealer / faulty weight profit problems

### 3. Ratio & Proportion

**5 Core Concepts**
1. A ratio a:b means a/b — it expresses relative magnitude, not absolute; always reduce to lowest terms by dividing by GCD
2. Proportion: a:b :: c:d means a/b = c/d, equivalently `ad = bc` (product of extremes = product of means)
3. Compounded ratio: a:b compounded with c:d = ac:bd — multiply numerators and denominators separately
4. Inverse ratio: if ratio is a:b, inverse ratio is b:a — used in time-work and speed problems where relationship is reciprocal
5. Dividing a quantity in a given ratio: if total T is split as a:b, first part = T × a/(a+b), second = T × b/(a+b)

**5 Tricks**
1. **Formula Shortcut — k-method:** Set ratio a:b as ak and bk; sum = (a+b)k; find k from given sum then compute parts directly
2. **Mental Math — Equivalent ratios:** Instead of dividing, multiply both sides so one value matches a given number; e.g. 3:5 with first part 12 → multiply by 4 → 12:20
3. **Formula Shortcut — Mean proportion:** Mean proportion of a and b is √(ab); third proportion of a and b is b²/a
4. **Trick — Ratio of mixtures:** If two solutions at ratio r1:r2 are mixed, result ratio determined by weighted sum; use alligation rule for fastest solve
5. **Mental Math — Comparison of ratios:** Cross-multiply to compare a/b vs. c/d: if ad > bc then a/b > c/d — avoid decimal conversion

**5 Problem Types**
1. Divide a total amount in a given ratio between 2 or 3 people
2. Find original amounts when ratio and sum/difference are given
3. Ages in ratio — find ages after or before N years
4. Income and expenditure in ratio — find savings
5. Mixture ratio — combine two ratios to find overall composition

### 4. Time, Speed & Distance

**5 Core Concepts**
1. Fundamental relationship: `Speed = Distance / Time`; the triangle — cover one to get formula for the other two; always check units match
2. Average speed for equal distances: `Avg Speed = 2S1S2 / (S1 + S2)` — NOT the arithmetic mean; this is the harmonic mean of two speeds
3. Relative speed: same direction → speeds subtract; opposite direction → speeds add; used for trains, boats, races
4. Trains: when train crosses a pole/person — distance = length of train; when it crosses a platform — distance = length of train + length of platform
5. Boats & Streams: downstream speed = Boat + Stream; upstream speed = Boat − Stream; `Boat speed = (D+U)/2`, `Stream speed = (D−U)/2`

**5 Tricks**
1. **Formula Shortcut — Average speed (equal distance):** 2ab/(a+b) — never average directly; this is the single most tested TSD trick
2. **Formula Shortcut — Relative speed (trains meeting):** Time to cross = (L1+L2) / (S1+S2) for opposite; (L1+L2) / (S1−S2) for same direction
3. **Mental Math — Unit conversion:** 1 km/h = 5/18 m/s; 1 m/s = 18/5 km/h; memorize these to convert instantly without calculation
4. **Formula Shortcut — Race:** If A beats B by d metres in a race of L metres: ratio of speeds = L : (L−d)
5. **Trick — Boats:** If downstream time is t1 and upstream is t2, then still-water speed = D(t1+t2)/(2t1t2); stream = D(t2−t1)/(2t1t2)

**5 Problem Types**
1. Find distance/time/speed given two of the three
2. A and B start from opposite ends — when do they meet, and where?
3. Train crossing problems (pole, platform, another train)
4. Boat and stream — find speed of boat or current
5. A covers distance at speed S1 but is late/early; find actual speed or distance

### 5. Time & Work

**5 Core Concepts**
1. Work rate principle: if A does a job in n days, A's one-day work = 1/n; add rates when working together: combined rate = 1/A + 1/B
2. LCM method: assume total work = LCM of all given days; compute daily work for each person as integers — avoids fractions entirely
3. Pipes & Cisterns analogy: inlet pipes fill (positive rate), outlet pipes drain (negative rate); net rate = sum with signs
4. Work equivalence: M1 × D1 × H1 = M2 × D2 × H2 (men × days × hours = constant for same total work); extend with efficiency if given
5. Leaving mid-way: if A works for x days then B joins (or leaves), split the work into two phases and set up equation for total = 1

**5 Tricks**
1. **Formula Shortcut — Two people together:** Time = AB/(A+B) where A and B are number of days each takes alone; memorize for 2-person problems
2. **LCM Method:** Total work = LCM(A's days, B's days, ...); each person's rate = LCM/their days; add rates; time = total/combined rate
3. **Formula Shortcut — One leaves early:** If A and B together finish in T days and A leaves after x days, work done by A = x/A; remaining = 1 − x/A done by B alone
4. **Formula Shortcut — Efficiency ratio:** If A is k times as efficient as B, A takes (1/k) the time of B; they together take B's time × k/(k+1)
5. **Mental Math — Pipes:** If pipe fills in F hours and drains in D hours, net fill time = FD/(D−F); if D>F tank never fills — check sign first

**5 Problem Types**
1. A and B working together — in how many days do they finish?
2. A and B start together; after x days one leaves — when does the other finish?
3. Pipe inlet and outlet — find fill/drain time with both open
4. M men complete work in D days; how many men needed to finish in D' days?
5. A is twice/thrice as efficient as B — find combined time or individual time

### 6. Number System

**5 Core Concepts**
1. Divisibility rules: 2 (last digit even), 3 (digit sum ÷3), 4 (last 2 digits ÷4), 5 (ends 0 or 5), 6 (÷2 and ÷3), 8 (last 3 digits ÷8), 9 (digit sum ÷9), 11 (alternate digit diff ÷11)
2. HCF and LCM: HCF × LCM = Product of two numbers (for exactly two numbers only); for HCF of fractions: HCF(numerators)/LCM(denominators)
3. Prime factorization: express as product of prime powers; number of factors = product of (power+1) for each prime; sum of factors formula
4. Remainders: use property that (a × b) mod m = ((a mod m) × (b mod m)) mod m; cyclicity of last digits (powers of 2,3,4 repeat in cycles)
5. Integers: natural numbers, whole numbers, integers, rational vs. irrational; even × odd rules; sum of first n naturals = n(n+1)/2; squares = n(n+1)(2n+1)/6

**5 Tricks**
1. **Divisibility shortcut — 7:** Double the last digit and subtract from rest; repeat until small; e.g. 161: 16 − 2 = 14 (÷7) ✓
2. **Last digit cyclicity:** Powers of 2 → 2,4,8,6 (cycle 4); powers of 3 → 3,9,7,1 (cycle 4); powers of 7 → 7,9,3,1; find power mod 4 then read cycle
3. **Formula Shortcut — Number of zeros in n!:** Sum of ⌊n/5⌋ + ⌊n/25⌋ + ⌊n/125⌋ + ... (count factors of 5; there are always more 2s)
4. **Mental Math — Sum of AP:** n/2 × (first + last); or n/2 × (2a + (n−1)d); recognize arithmetic sequences quickly in series questions
5. **Shortcut — Remainder of large powers:** Find pattern of remainders for base^1, base^2, ... mod divisor; cycle repeats; use power mod cycle-length

**5 Problem Types**
1. Find HCF/LCM of given numbers; application to real-world (tiles, bells ringing)
2. Find remainder when large expression is divided by a number
3. Find last digit of a^n for large n
4. Number of trailing zeros in n!
5. Find the number given properties about divisibility or digit sum

### 7. Average

**5 Core Concepts**
1. Average = Sum / Count; equivalently Sum = Average × Count — the second form is more useful in most problems
2. Weighted average: if group 1 has n1 items with average a1 and group 2 has n2 items with average a2, combined = (n1×a1 + n2×a2)/(n1+n2)
3. Effect of adding/removing a term: new sum = old sum ± new term; new average = new sum / new count
4. Effect of wrong entry: if a value was recorded wrongly, correct sum = wrong sum − wrong value + correct value; recalculate average
5. Average of consecutive numbers: average = (first + last)/2; for n consecutive integers starting at a, avg = a + (n−1)/2

**5 Tricks**
1. **Assume and adjust:** Assume average is a round number; for each value, note deviation from assumed average; true average = assumed + (sum of deviations)/count
2. **Shortcut — Entry/exit average effect:** If a person with value x replaces one with value y, new average = old average + (x−y)/n
3. **Shortcut — nth term from average change:** If average of n terms is A and adding one more makes it A', new term = A' × (n+1) − A × n
4. **Mental Math — Range check:** Average must lie between min and max; eliminate answer choices outside this range instantly
5. **Shortcut — Missing observation:** Given overall average and average of rest, missing = total sum − sum of rest

**5 Problem Types**
1. Average of a group; find sum or a missing term
2. A new member joins / one leaves; find change in average
3. Weighted average of two groups combined
4. Wrong entry corrected; find correct average
5. Average of first n natural numbers / even numbers / odd numbers

### 8. Simple Interest & Compound Interest

**5 Core Concepts**
1. Simple Interest: `SI = PRT/100`; Amount = P + SI = P(1 + RT/100); interest is same every period (linear growth)
2. Compound Interest: `A = P(1 + R/100)^T`; CI = A − P; interest is on principal + accumulated interest (exponential growth)
3. Compounding frequency: half-yearly → rate = R/2, time = 2T; quarterly → R/4, time = 4T
4. Difference between CI and SI for 2 years: `CI − SI = P(R/100)²`; for 3 years: `= P(R/100)² × (R/100 + 3)`
5. Effective rate: when compounded half-yearly at R%, effective annual rate = (1 + R/200)² − 1; always more than R%

**5 Tricks**
1. **Formula Shortcut — SI rate from installments:** Memorize SI formulas for equal installment repayment
2. **Formula Shortcut — CI for 2 years shortcut:** A = P × (1 + R/100)²; expand as P + 2PR/100 + PR²/10000; the last term is the extra over SI
3. **Mental Math — Doubling time:** At R% SI, money doubles in 100/R years; at R% CI, roughly 72/R years (Rule of 72)
4. **Shortcut — Population growth:** Same formula as CI; P(1 + R/100)^T for growth; P(1 − R/100)^T for decay
5. **Formula Shortcut — CI−SI difference (2 yr):** = P × (R/100)²; fastest way to find difference without computing both separately

**5 Problem Types**
1. Find SI/CI given P, R, T
2. Find P or R or T when SI/CI and two others are given
3. Find CI when compounded half-yearly or quarterly
4. Find difference between CI and SI for 2 or 3 years
5. Population / depreciation problems using CI formula

### 9. Mixture & Alligation

**5 Core Concepts**
1. Alligation rule: the ratio in which two ingredients at prices C1 and C2 must be mixed to get mixture at price Cm is `(C2 − Cm) : (Cm − C1)`; draw the cross
2. Mean price (Cm) must lie between C1 and C2; if it doesn't, the problem is set up incorrectly
3. Repeated dilution: if from V litres of pure liquid, x litres are removed and replaced with water n times, pure liquid remaining = V × ((V−x)/V)^n
4. Mixing two mixtures: if mixture 1 has A:B ratio r1:s1 and mixture 2 has r2:s2, combined ratio found by weighted contribution of each component
5. Percentage-based alligation: can apply to any two values (% alcohol, % profit, % marks) — not just price; Cm = weighted average = target%

**5 Tricks**
1. **Alligation Cross:** Write C1 top-left, C2 top-right, Cm centre; diagonals give differences; ratio = bottom-right : bottom-left
2. **Shortcut — Repeated replacement:** Remaining pure = V × ((V−x)/V)^n — memorize; don't compute step by step
3. **Mental Math — Ratio check:** Sum of alligation ratio = total parts; scale to actual quantities given
4. **Shortcut — Milk & water:** If ratio of milk to water in final mixture is m:w, then milk% = m/(m+w) × 100; work backwards to find initial
5. **Formula Shortcut — Average in alligation:** When mixing n1 units at p1 and n2 units at p2: avg = (n1×p1 + n2×p2)/(n1+n2); set equal to given avg to find ratio

**5 Problem Types**
1. In what ratio should two items at prices P1 and P2 be mixed to sell at price Pm?
2. From a container of pure liquid, x litres removed and replaced with water — find concentration after n replacements
3. Two mixtures with different compositions mixed — find resulting ratio
4. Given final ratio and one quantity, find the other using alligation
5. A shopkeeper mixes two varieties of rice — find cost price or selling price for target profit

### 10. Mensuration

**5 Core Concepts**
1. 2D shapes — commit to memory: Triangle (½bh; Heron's: √s(s−a)(s−b)(s−c) where s=(a+b+c)/2); Circle (πr², 2πr); Rectangle (lb, 2(l+b)); Parallelogram (bh); Trapezium (½(a+b)h)
2. 3D shapes: Cube (a³ vol, 6a² surface); Cuboid (lbh, 2(lb+bh+lh)); Cylinder (πr²h vol, 2πrh+2πr²); Cone (⅓πr²h, πrl+πr²); Sphere (4/3πr³, 4πr²)
3. Slant height of cone: l = √(r²+h²); used in lateral surface area; always compute before surface area
4. Scaling rule: if all linear dimensions scaled by factor k, area scales by k², volume scales by k³
5. Path/border problems: outer area − inner area; outer dimensions = inner + 2×width of border

**5 Tricks**
1. **Memory device — Cone vs Pyramid:** Cone = circular base (πr²h/3); Pyramid = rectangular/square base (Bh/3 where B = base area)
2. **Shortcut — Diagonal of rectangle:** d = √(l²+b²); for square d = a√2; for cube d = a√3
3. **Shortcut — Area of equilateral triangle:** = (√3/4)a²; perimeter = 3a; height = (√3/2)a
4. **Mental Math — π approximation:** Use 22/7 when r involves 7-multiples; use 3.14 for decimals; choose based on cleaner calculation
5. **Scaling trick:** If radius doubles, area quadruples (×4), volume multiplies by 8 (×2³); saves re-calculation from scratch

**5 Problem Types**
1. Find area/perimeter/volume of a standard shape given dimensions
2. Cost of fencing/flooring/painting — area × rate
3. Cylinder/cone/sphere conversions (e.g., melting and recasting)
4. Combined shapes (hemisphere on cylinder; cone on cylinder)
5. Scale change — if dimension increases by x%, find % change in area or volume

### 11. Algebra

**5 Core Concepts**
1. Linear equations: one variable `ax + b = 0`; two variables — solve by substitution or elimination; check for unique/no/infinite solutions (consistent vs. inconsistent)
2. Quadratic equations: `ax² + bx + c = 0`; roots = `(−b ± √(b²−4ac)) / 2a`; sum of roots = −b/a; product of roots = c/a
3. Identities to memorize: (a+b)² = a²+2ab+b²; (a−b)² = a²−2ab+b²; (a+b)(a−b) = a²−b²; (a+b)³ = a³+3a²b+3ab²+b³; a³+b³ = (a+b)(a²−ab+b²)
4. Polynomial division and remainder theorem: f(a) gives remainder when f(x) divided by (x−a); if f(a)=0, (x−a) is a factor
5. Inequalities: multiplying/dividing by negative flips the inequality sign; for absolute value |x| < a means −a < x < a

**5 Tricks**
1. **Shortcut — Sum/product of roots:** Without solving quadratic, know sum = −b/a and product = c/a; use to find expressions like α²+β², αβ(α+β), etc.
2. **Identity trick:** If x + 1/x = k, then x² + 1/x² = k² − 2; x³ + 1/x³ = k³ − 3k; chain applies repeatedly
3. **Mental Math — Factorization:** For quadratics with integer roots, find two numbers whose sum = b and product = ac (split the middle term)
4. **Shortcut — Age/number problems:** Assign single variable; write all relationships; solve one equation; avoid multi-variable whenever possible
5. **Formula Shortcut — (a+b+c)²:** = a²+b²+c²+2(ab+bc+ca); if a+b+c=0, then a³+b³+c³ = 3abc — often tested directly

**5 Problem Types**
1. Solve linear equation in one or two variables; find the value
2. Find roots of a quadratic; determine nature of roots using discriminant
3. Apply algebraic identities to simplify or find missing value
4. Word problems — age, numbers, fractions — converted to equations
5. Remainder theorem — find remainder when polynomial divided by linear factor

### 12. Geometry

**5 Core Concepts**
1. Triangle properties: sum of angles = 180°; exterior angle = sum of two non-adjacent interior angles; sides: largest angle opposite longest side; triangle inequality: sum of any two sides > third
2. Special triangles: equilateral (all 60°); isosceles (base angles equal); right-angled (Pythagoras: a²+b²=c²); 30-60-90 ratio 1:√3:2; 45-45-90 ratio 1:1:√2
3. Circle theorems: angle in semicircle = 90°; angles subtended by same arc are equal; angle at centre = twice angle at circumference; tangent ⊥ radius at point of contact
4. Parallel lines and transversals: corresponding angles equal; alternate interior angles equal; co-interior (same-side) angles supplementary (sum = 180°)
5. Similarity and congruence: similar triangles have equal angles and proportional sides; AAA, SAS, SSS similarity criteria; ratio of areas = square of ratio of corresponding sides

**5 Tricks**
1. **Pythagorean triplets:** 3-4-5, 5-12-13, 8-15-17, 7-24-25, 9-40-41; recognize scaled versions (6-8-10, 10-24-26); saves computation time
2. **Shortcut — Median to hypotenuse:** In right triangle, median to hypotenuse = half the hypotenuse; elegant shortcut for such problems
3. **Trick — Tangent from external point:** Two tangents from external point to circle are equal in length; draw auxiliary lines to form right triangles
4. **Mental Math — Angle in polygon:** Sum of interior angles of n-gon = (n−2)×180°; each interior angle of regular n-gon = (n−2)×180°/n
5. **Shortcut — Similar triangles ratio:** If sides in ratio k:1, areas in ratio k²:1; perimeters in ratio k:1; use area ratio to back-calculate k

**5 Problem Types**
1. Find unknown angle using triangle properties or parallel lines
2. Find side or area using Pythagoras or similar triangles
3. Circle — find chord length, arc, angle, or area of sector/segment
4. Quadrilateral properties (parallelogram, rhombus, trapezium) — find area or missing angle
5. Combination question — circle inscribed in triangle or vice versa; find radius or side

### 13. Trigonometry

**5 Core Concepts**
1. Basic ratios: sin θ = P/H; cos θ = B/H; tan θ = P/B; reciprocals: cosec, sec, cot; SOHCAHTOA mnemonic
2. Standard values table — must memorize: sin/cos/tan for 0°, 30°, 45°, 60°, 90°; e.g. sin30°=½, cos60°=½, tan45°=1, sin90°=1
3. Pythagorean identities: sin²θ + cos²θ = 1; 1 + tan²θ = sec²θ; 1 + cot²θ = cosec²θ — derive from each other
4. Complementary angles: sin θ = cos(90−θ); tan θ = cot(90−θ); sec θ = cosec(90−θ) — used to simplify expressions
5. Heights and distances: angle of elevation (looking up); angle of depression (looking down); tan(angle) = height / horizontal distance; standard setup

**5 Tricks**
1. **Memory — Values table:** Row pattern for sin: 0°→√0/2, 30°→√1/2, 45°→√2/2, 60°→√3/2, 90°→√4/2; cos is reverse of sin row
2. **Identity manipulation:** Express everything in sin and cos; cancel; use identities to reduce expression to a constant or simpler form
3. **Complementary shortcut:** sin10°·sin80° = sin10°·cos10°; product pairs that add to 90° simplify via complementary rule
4. **Shortcut — Two buildings problem:** If angles of elevation from same point to two buildings are θ1 and θ2, use tan to set up two equations; solve simultaneously
5. **Mental Math — 45° symmetry:** At 45°, sin=cos=1/√2; tan=cot=1; sec=cosec=√2; any expression evaluated at 45° becomes equal-value pairs

**5 Problem Types**
1. Find the value of a trigonometric expression using identities
2. Prove a trigonometric identity
3. Find height or distance using angle of elevation/depression
4. Find value when one ratio is given (e.g., if sinθ = 3/5, find cosθ and tanθ)
5. Two-step heights and distances (flagpole on building; shadow problems)

### 14. Data Interpretation

**5 Core Concepts**
1. Table DI: read row and column headers carefully before calculating; percentage = (part/whole)×100; always identify what the "total" is
2. Bar chart DI: approximate from bar heights; ratio questions don't require exact values — compare visually first
3. Pie chart DI: each degree = total/360; sector value = (angle/360)×total; percentage = angle/3.6
4. Line graph DI: percentage change = (new−old)/old × 100; trend questions require spotting max/min change between consecutive periods
5. Caselets (paragraph DI): convert text to a table first; label all extracted numbers; identify what's given vs. what's computed

**5 Tricks**
1. **Approximation rule:** In DI, options are usually spread wide enough that rounding to nearest 5 or 10 still gives unique correct answer; calculate approximately, don't compute exactly
2. **Ratio shortcut:** When asked "what fraction of X is Y", set up ratio without computing absolute values if both read from same chart scale
3. **Percentage change check:** If decrease, answer < original; if increase, answer > original; eliminate wrong answer choices instantly by sign check
4. **Common denominator in pie:** All pie DI shares the same total; compute total once; reuse across multiple questions in the set
5. **Time management:** Scan all 4-5 questions of a DI set before solving; pick the easy 2-3 first; skip the hard calculation-heavy one if stuck

**5 Problem Types**
1. Find the percentage share or absolute value for a specific category/year
2. Find the ratio between two categories or two years
3. Find the percentage change from one period to the next
4. Find the total or combined value across multiple categories
5. Find rank or comparison (which category is highest/lowest/second highest)

### 15. Statistics & Probability

**5 Core Concepts**
1. Central tendency: Mean = sum/n; Median = middle value (sorted); Mode = most frequent value; for even n, median = average of two middle values
2. Deviation: Range = max − min; Variance = average of squared deviations from mean; Standard deviation = √Variance; higher SD = more spread
3. Probability: P(E) = favourable outcomes / total outcomes; 0 ≤ P(E) ≤ 1; P(E') = 1 − P(E); P(A∪B) = P(A) + P(B) − P(A∩B)
4. Independent events: P(A and B) = P(A) × P(B); mutually exclusive: P(A or B) = P(A) + P(B); conditional: P(A|B) = P(A∩B)/P(B)
5. Permutations and Combinations: nPr = n!/(n−r)!; nCr = n!/((n−r)!×r!); arrangements = permutations (order matters); selections = combinations (order doesn't matter)

**5 Tricks**
1. **Median shortcut:** Sort the list first; for odd n, position = (n+1)/2; for even n, average positions n/2 and n/2+1
2. **Probability complement:** P(at least one) = 1 − P(none); much easier for complex events; compute the "none" case and subtract
3. **Dice/coin patterns:** 2 dice: total outcomes = 36; for sum s, count pairs manually; coins: 2 coins → 4 outcomes; n coins → 2^n outcomes
4. **Combination identity:** nCr = nC(n−r); use whichever gives smaller number to compute; e.g. 10C8 = 10C2 = 45
5. **Mean shift:** Adding constant k to all values shifts mean by k but doesn't change variance; multiplying all by k multiplies mean by k and variance by k²

**5 Problem Types**
1. Find mean, median, mode of a given dataset
2. Probability of a single event (cards, dice, balls in bag)
3. Probability of combined events (both A and B; at least one of A or B)
4. Number of arrangements or selections (permutations and combinations)
5. Find missing value when mean/median is given and one term is unknown
